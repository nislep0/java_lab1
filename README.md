# Лабораторна робота №1

## Тема
Масиви в мові програмування Java

## Мета роботи
Ознайомлення з масивами та використання основних методів їх обробки в мові програмування Java.  
Набуття практичних навичок роботи з двовимірними масивами (матрицями), виконання матричного добутку та обробки результуючої матриці.

## Варіант
Мій номер у списку групи - 19:
- C5 = 19 mod 5 = 4
- C7 = 19 mod 7 = 5
- C11 = 19 mod 11 = 8

## Завдання
1. Створити дві матриці `A` та `B` з типом елементів `char`.
2. Виконати операцію матричного добутку:
    ```bash
    C = A × B
    ```
3. Для результуючої матриці `C` знайти середнє значення елементів кожного рядка.
4. Реалізувати програму мовою Java з використанням масивів.
5. Вивести на екран матриці `A`, `B`, `C` та результати обчислення середніх значень.
6. Обробити можливі виключні ситуації.
7. Дотримуватися стандартів Java Code Conventions.

## Теоретичні відомості

Матриця в Java може бути представлена двовимірним масивом.  
Матричний добуток можливий лише за умови, що кількість стовпців матриці `A` дорівнює кількості рядків матриці `B`.  
Оскільки тип `char` у Java є числовим типом і зберігає Unicode символу, при виконанні арифметичних операцій використовується його числове значення.  
Середнє значення елементів рядка обчислюється як сума числових значень символів, поділена на кількість елементів у рядку.

## Опис програми

Програма складається з одного класу `lab1`, який містить:
- виконавчий метод `main`;
- метод для множення матриць;
- метод для обчислення середніх значень рядків;
- метод для виведення матриці на екран.

Усі змінні ініціалізуються у методі `main`.  
У програмі передбачена обробка виключних ситуацій, зокрема перевірка можливості множення матриць.

## Результати роботи програми

- Виведено матриці A, B та результуючу матрицю C.
- Для кожного рядка матриці C обчислено середнє значення елементів.
- Програма коректно обробляє помилки несумісності розмірів матриць.
